Speluitleg

  1. Je bent een architect in een organisatie die een toekomstbestendige Enterprise Architecture moet neerzetten volgens de Dragon1-methodiek.
  2. Kies een rol: Enterprise / Solution / Business / Information / Security Architect, CIO, IT-Manager of Student.
  3. Verplaats over het 4×4 bord van 16 organisatie-onderdelen (Directiekamer → Governance Board).
  4. Op elke locatie kun je een missie trekken. Missies leveren Dragon1-artefacten op (Vision Blueprint, Strategy Map, Capability Map, Application Landscape, …).
  5. Onderweg krijg je uitdagingen (Legacy, ShadowIT, NIS2, Ransomware, Fusie, …). Je kiest hoe je erop reageert — elke keuze heeft impact op je 8 kernmetrieken.
  6. Events zijn neutraal/gemengd, vaak met branching-keuzes.
  7. Je 12 dashboard-metrieken: Level · XP · Business Value · Stakeholder Confidence · Governance · Innovation · Security · Technical Debt · Budget · Digital Maturity · Lopende missie · Artefacten.
  8. Verzamel de 12 kern-Dragon1-artefacten — sommige hebben cascade-vereisten (bv. Application Landscape vereist Capability Map).
  9. Wanneer klaar: klik Presenteer aan RvB. Je ontvangt een evaluatierapport op 8 aspecten (Samenhang / Strategie-uitlijning / Businesswaarde / Innovatie / Governance / Security / Digitale volwassenheid / Stakeholdertevredenheid) met verbeterpunten.
  10. Autosave staat aan. Kies "Doorgaan" op het startscherm om terug te keren in je workshop.
